Virtual Embryo: About

This page was created by Jake Warner to enable visual exploration of gene territories in the Nematostella vectensis embryo. The website is written completely in javascript using the three.js library for modeling.

Data sources

Spatial (in situ hybridization) expression data are from the gene expression database at KahiKai.org. RNAseq data are from the transcriptomic databases found at NvERTx.
